# Checkpoints

Place external weights in this directory before running training or inference.

Required assets:

- `dinov3_vitb16_pretrain_lvd1689m-73cec8be.pth`
- `dinov3_vitl16_pretrain_lvd1689m-8aa4cbdd.pth`
- `model_final_e96218.pkl`
- `phi_smpl_27554_256.pkl`

Optional assets:

- Your trained static checkpoints, for example `static_vitl.pt`
- Your trained dynamic checkpoints, for example `dynamic_vitl.pt`

This folder is intentionally empty in the open-source release.
